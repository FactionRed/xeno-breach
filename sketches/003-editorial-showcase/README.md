## Variant: Editorial Showcase

### Design stance
Magazine-style alternating feature blocks — each game gets a full editorial spread with art and copy.

### Key choices
- Layout: Centered hero → alternating text/art rows (like a product page)
- Typography: Georgia serif for body, sans-serif for UI elements
- Color: Warm cream background, minimal — lets content breathe
- Interaction: Smooth scroll, download/GitHub buttons, responsive

### Trade-offs
- Strong at: Telling a story per game, premium feel, mobile-friendly
- Weak at: Quick scanning — each game takes a full viewport

### Best for
- Studios with 1-5 games where each title deserves narrative space
