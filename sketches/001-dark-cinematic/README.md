## Variant: Dark Cinematic

### Design stance
Full-bleed hero with the featured game front and center, dark sci-fi aesthetic matching the game's visual identity.

### Key choices
- Layout: Hero → card grid (like Steam/GOG)
- Typography: System sans-serif, bold uppercase title
- Color: Near-black background, cyan + red accents (matches game palette)
- Interaction: Clickable cards, GET button on hover, nav links

### Trade-offs
- Strong at: Showcasing a flagship title, immersive feel
- Weak at: Browsing many games at once (only 3 cards visible)

### Best for
- Studios with 1-3 games where the hero title matters more than catalog depth
