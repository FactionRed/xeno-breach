## Variant: Compact Catalog

### Design stance
Dense, browsable catalog with sidebar filters and search — like itch.io or a lightweight Steam.

### Key choices
- Layout: Sidebar + grid (catalog-first)
- Typography: System sans-serif, small and dense
- Color: Light gray background, dark sidebar, green/red price badges
- Interaction: Working search, category filters, sort buttons, clickable cards

### Trade-offs
- Strong at: Browsing many games, filtering by genre/price
- Weak at: Emotional impact — feels utilitarian, not cinematic

### Best for
- Studios with 5+ games where discovery and filtering matter
